# shoppingcart

## Getting Started
Icon을 버튼화 시키는 방법 : Inkwell, IconButton 두가지 방법.
Inkwell은 행위를 onTap
IconButton은 행위를 InPressed

앱의 오른쪽 debug 없애는 방법 : debugShowCheckedModeBanner: false 

여백을 줄 때 sizedBox를 선호 하는 이유 
그냥 단지 Padding값을 선언하면 재활용 할 때 Padding값이 
없어서 재활용이 불가능할 수 있다.
그래서 sizedBox를 준다.
